There are two models running locally. First detects possible squares that can form sudoku table, while the other one try to find digits.
App allows user both import images manually or scan the screen. There is no any violation of provacy, everything happens inside app itself, locally.
So, screen recording only happens after permission, after permission granted app follows screen and shot when user taps scan button. There is no any back end integration so user can use app wihtout any concern.
